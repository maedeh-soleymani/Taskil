# Taskil

Taskil is a fast and intuitive task management app that helps you capture your thoughts instantly. With AI-powered insights, it prioritizes your tasks and shows you which ones need your attention the most, keeping you focused and organized at all times.

## Features
- Quick task capture to clear your mind
- AI-assisted task prioritization
- Simple and minimal interface
- Always know what to do next
